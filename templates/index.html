THE APPLICATION SUPPORTS THE FOLLOWING FILE FORMATS:

  • CSV - text files with delimiters
  • Excel - tables in XLSX format
  • SAS7BDAT - standard SAS files
  • XPT - standard SAS transport files

Upload a File:

Please note:

◾️ Unfortunately, CPORT files created using PROC CPORT in SAS are not supported. This is because the format is proprietary and not supported by most open-source Python libraries.

If you have a file in CPORT format, you can:

  • Use SAS or WPS to convert it to SAS7BDAT or XPT format.
  • Use a commercial tool such as Stat/Transfer.

◾️ We recommend converting files to a supported format before uploading them. If you have any questions, please contact our technical support team.
